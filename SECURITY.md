# Security Policy

## Supported versions

Only the latest minor of the current major receives security fixes.
While `speechnorm` is in `v0.x.y`, only the latest minor is supported.

## Reporting a vulnerability

Please report security issues privately via GitHub Private Vulnerability
Reporting:

https://github.com/OliDow/speechnorm/security/advisories/new

We aim to acknowledge reports within 72 hours.

Please do not open public issues for security problems.
